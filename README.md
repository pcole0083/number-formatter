Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @zairon87/number-formatter`

## Usage

    var numFormatter = require('@zairon87/number-formatter');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`
